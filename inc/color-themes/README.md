# Backbone Theme for SEO + LLMO

[![Version](https://img.shields.io/badge/Version-1.0.12--dev-orange.svg)](https://github.com/TsuyoshiKashiwazaki/wp-theme-backbone-seo-llmo/releases)
[![License](https://img.shields.io/badge/License-GPL%20v2-blue.svg)](https://www.gnu.org/licenses/gpl-2.0.html)
[![WordPress](https://img.shields.io/badge/WordPress-5.0%2B-blue.svg)](https://wordpress.org/)

メディア運用に特化したシンプルなWordPressベーステーマ。基本的なSEO対応とレイアウト機能を提供し、高度な機能は専用プラグインでの拡張を前提とした設計。

## Features

- 基本的なSEO機能（メタディスクリプション、タイトルタグ最適化）
- 柔軟なレイアウトシステム（1/2/3カラム、フルワイド）
- 22個のWCAG AAA準拠カラーテーマ（系外惑星名）
- カスタマイザー統合
- レスポンシブデザイン
- 開発者向けキャッシュバスティング機能

## Requirements

- WordPress 5.0+
- PHP 7.2+

## Installation

1. このリポジトリをクローンまたはダウンロード
2. `wp-content/themes/` にアップロード
3. WordPress管理画面でテーマを有効化
