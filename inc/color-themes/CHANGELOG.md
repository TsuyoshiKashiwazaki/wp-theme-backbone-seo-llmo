# Changelog

## [1.0.12] - 2025-11-03

### 削除
- WordPress Core「色」セクションをカスタマイザーから削除
  - `add_theme_support('custom-background')` 削除
  - `backbone_remove_colors_section()` 追加で強制削除

### 追加
- 新22カラーテーマ（11色 x 明暗）系外惑星名で作成
  - ケプラー22b（青）、ベテルギウス（赤）、ルイテン726（緑）
  - ワスプトゥエルブ（黄）、プサイドラコニス（紫）
  - アルゴラブ（蛍光オレンジ）、アルゴラブK0V（オレンジ+黒ヘッダー）
  - アークトゥルス（橙）、コロート7b（灰）、デネブ（白）
  - シリウスb（黒）、カペラ（クリーム）
- 全テーマWCAG AAA準拠、色覚多様性対応
- リンクに細く薄い下線を追加（0.5px、視認性向上）

### 変更
- 旧40+カラーテーマを `inc/color-themes/old/` ディレクトリに移動（保存済み）
- ヘッダーサブタイトルの色を白色に変更
- フッター全テキスト・リンクを白色に変更
- カード背景色・境界線をCSS変数化
- カード内テキスト色をCSS変数化
- ヘッダー・フッターの配色を完全統一
- フッターグラデーションバーをprimary-colorに統一

### 修正
- ヘッダーサブタイトルが見えない問題
- フッターテキストが見えない問題
- 暗いテーマでtext_lightが暗すぎる問題
- カードテキストが読めない問題
- カテゴリバッジ・アーカイブリンクの視認性問題

### 技術的改善
- ハードコードされた色を全てCSS変数化
- front-page-sections.css: 7箇所の色をCSS変数化
- components-footer.css: 全テキスト要素の色を統一
- header.php: サブタイトルにinline style追加
- base.css: リンクに下線スタイル追加
