メディアサイト向けに最適化された軽量WordPressベーステーマ。基本的なSEO機能を内蔵し、プラグインによる拡張を前提とした設計。

== 説明 ==

Backbone Theme for SEO + LLMO は、メディア運用に特化したシンプルなWordPressベーステーマです。基本的なSEO機能を提供しつつ、高度な機能は専用プラグイン（SEO・LLMO・GEO・AIO最適化プラグインなど）での拡張を前提とした設計です。

= 主な機能 =

* **プラグイン優先設計** - 専用プラグインによる拡張を前提とした設計
* **WCAG AAA準拠カラーテーマ** - 22種類の系外惑星・恒星テーマのアクセシブルカラースキーム
* **基本SEO対応** - メタディスクリプション抽出、タイトル最適化
* **柔軟なレイアウト** - 1〜3カラムレイアウト、フルワイド対応
* **カスタマイザー連携** - カラー、レイアウト、タイポグラフィ、デザイン調整
* **メディア最適化** - ブログ・メディアサイト向けに最適化
* **軽量設計** - 必要最小限の機能のみ搭載
* **レスポンシブ** - モバイルファースト設計
* **アクセシビリティ重視** - WCAG AAA準拠、色覚多様性対応

== インストール ==

= 自動インストール =

1. WordPressダッシュボードから「外観」→「テーマ」→「新規追加」を選択
2. 「Backbone Theme for SEO + LLMO」を検索
3. 「インストール」をクリックし、「有効化」

= 手動インストール =

1. テーマのZIPファイルをダウンロード
2. WordPressダッシュボードから「外観」→「テーマ」→「新規追加」→「テーマのアップロード」を選択
3. ZIPファイルを選択してインストール
4. テーマを有効化

= セットアップ =

1. 「外観」→「カスタマイズ」でカラー、レイアウト、タイポグラフィを調整
2. 「外観」→「メニュー」でメインメニューとフッターメニューを作成
3. 「外観」→「ウィジェット」でサイドバーとフッターにウィジェットを追加
4. 投稿と固定ページにアイキャッチ画像を設定

== よくある質問 ==

= このテーマはどのようなサイトに適していますか？ =

ブログ、メディアサイト、ニュースサイトなど、コンテンツ重視のウェブサイトに最適です。シンプルで拡張性のある設計により、さまざまな目的に対応できます。

= プラグインなしでこのテーマを使用できますか？ =

はい、基本的なブログやウェブサイトとして問題なく機能します。ただし、高度なSEO、LLMO、GEO機能については専用プラグインの使用を推奨します。

= カラーテーマは何種類ありますか？ =

22種類のWCAG AAA準拠カラーテーマ（11色×2バリエーション）があります。すべてのテーマは色覚多様性に対応したコントラスト比を持ち、視覚障害のあるユーザーにも読みやすい設計です。

= カスタマイザーではどのような設定ができますか？ =

カラー設定（プリセットテーマ、カスタムカラー）、レイアウト設定（カラム数、サイドバー位置、コンテナ幅）、タイポグラフィ設定（フォントファミリー、サイズ、行間）、デザインパターン、装飾テーマ、ヘッダー/フッター設定、フロントページ設定、アーカイブ設定、フォーム設定、カスタムコードなど。

= 子テーマは必要ですか？ =

テーマをカスタマイズする予定がある場合は、子テーマの使用を強く推奨します。テーマの更新時にカスタマイズが失われることを防ぎます。

= 公式の子テーマサンプルはありますか？ =

はい、公式の子テーマサンプルを以下で提供しています:
https://github.com/TsuyoshiKashiwazaki/wp-theme-backbone-seo-llmo-child

子テーマには詳細なドキュメントとカスタマイズ例が含まれています。親テーマと子テーマの両方をインストールし、子テーマを有効化してください。

= モバイル対応していますか？ =

はい、このテーマはモバイルファーストのレスポンシブデザインを採用しており、スマートフォン、タブレット、デスクトップで最適に表示されます。

= 対応ブラウザは？ =

モダンブラウザの最新バージョン（Chrome、Firefox、Safari、Edge）をサポートしています。
