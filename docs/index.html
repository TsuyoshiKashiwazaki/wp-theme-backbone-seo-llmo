Backbone Theme for SEO + LLMO マニュアル

メディア運用に特化したシンプルなベーステーマです。基本的なSEO対応とレイアウト機能を提供し、高度な機能は専用プラグインでの拡張を前提とした設計になっています。

Backbone Theme は「テーマは基盤、プラグインで拡張」という思想のもと開発されました。カスタマイザーによる100以上の設定項目で、コードを書くことなくサイトの外観・レイアウト・コンテンツ表示を細かく調整できます。SEO基本機能（メタディスクリプション・メタキーワード自動生成）やLLMO対応機能（セマンティックHTML・カスタムJSON-LD）も内蔵しており、プラグインなしでも基礎的なSEO/LLMO対策が可能です。

設計思想

Backbone Theme は「テーマは基盤（Backbone）、プラグインで拡張」という思想に基づいています。テーマ単体でサイトの見た目・レイアウト・基本SEOを完結させ、高度なSEO・LLMO・GEO・AIO（AI Optimization）機能は専用プラグインで後付けする構造です。

この設計により、テーマ更新時にプラグインの機能が失われるリスクを排除し、テーマとプラグインの責任範囲を明確に分離しています。テーマが提供するのは「表示基盤」と「最低限のSEO」であり、構造化データの自動生成やAI検索向け最適化などの専門機能はプラグイン側に委譲します。

主要機能

柔軟なレイアウト

1カラム・2カラム・3カラム・フルワイドの4種類から選択。投稿タイプごとに個別設定が可能で、サイドバーの左右位置・幅（200〜500px）も変更できます。1カラム時のコンテンツ最大幅（600〜1920px）も調整可能です。

44種類のカラーテーマ

天体名をモチーフにした22種（各ライト/ダーク）のプリセットに加え、独自カラーテーマ機能で21のCSS変数（プライマリ、セカンダリ、アクセント、背景、テキスト、リンク、ボタン、フォーム等）を個別にカスタマイズできます。

SEO + LLMO対応

メタディスクリプション・メタキーワード自動生成、セマンティックHTML、タイトルセパレーター（|）、カスタムヘッダーコード（JSON-LD等）の投稿別入力に対応。AI検索エンジン（ChatGPT・Gemini・Perplexity等）からの引用精度も向上します。

フロントページビルダー

リストセクション5枠＋個別セクション5枠＋フリーコンテンツの組み合わせで、トップページを自由に構築。セクションの表示順はドラッグ&ドロップで変更できます。

メインビジュアル

標準・フルワイド・サークル・カード型の4スタイルに加え、ボーダー・角丸・アニメーション（フェードイン・スライドアップ・ズームイン等）・配置を投稿タイプ別に設定可能です。

サブディレクトリ別ブランディング

最大10個のサブディレクトリに対して、個別のロゴ・カラーテーマ・デザインパターン・タイポグラフィ・デコレーションを設定。1つのサイト内で複数のブランドを表現できます。

SEO / GEO / LLMO における強み

Backbone Theme は、従来の検索エンジン最適化（SEO）だけでなく、AI検索エンジンからの引用精度を高めるGEO（Generative Engine Optimization）やLLMO（Large Language Model Optimization）にも対応する基盤を提供します。

対策領域 テーマが提供する機能
メタ情報 メタディスクリプション自動生成（投稿・固定ページ・各種アーカイブ対応）、メタキーワード自動生成（focus_keyword・タグ・カテゴリ・タイトルから抽出）、タイトルセパレーター「|」
セマンティックHTML <article> / <header> / <nav> / <main> / <aside> / <footer> / <time> の適切な使用。クローラーとAIの両方がコンテンツ構造を正しく理解可能
構造化データ 投稿ごとのJSON-LD直接入力。Article / FAQPage / HowTo / Product / LocalBusiness / BreadcrumbList等のSchemaに対応
内部リンク構造 タクソノミー横断表示（カテゴリ・タグアーカイブに全投稿タイプの記事を統合表示）でサイト内の関連性を強化。タクソノミールートページで一覧ハブを提供
ページネーション /page-N/形式のSEOフレンドリーURL。旧形式は301リダイレクトで重複コンテンツを防止
